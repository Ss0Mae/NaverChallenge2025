### 코드 구조 개선 (리팩토링)

- [x] **모듈 설계**: `my-git.js`의 기능을 `GitRepository`, `ObjectManager`, `IndexManager`, `RefsManager` 등 역할 기반의 모듈로 분리하는 아키텍처 설계하기
- [x] **`ObjectManager` 구현**: 객체 읽기(`readObject`), 쓰기(`hashObject`) 관련 로직을 별도 모듈로 분리하기
- [x] **`IndexManager` 구현**: `readIndex`, `writeIndex`, `updateIndex` 등 인덱스 관련 로직을 별도 모듈로 분리하기
- [x] **`RefsManager` 구현**: `getHeadCommit`, `getHeadRef`, `updateHead` 등 참조 관련 로직을 별도 모듈로 분리하기
- [x] **CLI 진입점(`my-git-cli.js`) 재구성**: 각 모듈을 가져와 사용자의 명령어에 따라 실행 흐름을 제어하도록 `switch` 문 리팩토링하기
